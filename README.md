# IdeaTribe Frontend Assignment

This project is a frontend implementation of the provided Figma design as part of the IdeaTribe assignment.  
The goal was to translate the design into a clean, responsive, and modular landing page.

---

## Tech Stack

- React (Vite)
- Tailwind CSS (via CDN)
- JavaScript (ES6+)

---

## Features Implemented

- Separate **Header** and **Navigation Menu** as per requirements
- Center-aligned navigation with hover dropdowns
- Hero section with:
  - Right-aligned content
  - Fade-in animation
  - CTA grow animation
- Divider section with background image
- “Build, Validate, Grow” cards with scroll-based stacking effect
- Journey CTA section with background image and overlay
- Basic, clean footer
- Fully responsive layout

---

## Design Reference

The UI was implemented based on the Figma design shared via email.  
The Figma file was used strictly as a **design reference**, and the final submission is the React codebase.
